# z-index-list

Get a list of z-indexes used in your project.

Currently only `z-index` or `zIndex` properties are scanned for in the following file types:

- `.css`
- `.html`
- `.js`
- `.jsx`
- `.ts`
- `.tsx`

## Usage

```bash
npx z-index-list
pnpm dlx z-index-list
bunx z-index-list
# etc..
```
